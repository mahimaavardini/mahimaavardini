INVESTIGATING A COMPROMISED VM

What I Did

  • Conducted a full-scale forensic analysis of a Windows virtual machine compromised via PowerShell-based FTP exfiltration scripts.
  • Identified malicious persistence mechanisms through registry key modifications (RunOnce) and Windows Task Scheduler manipulations ensuring attacker foothold post-reboot.
  • Performed deep packet inspection in Wireshark to trace data exfiltration channels and validated FTP credential misuse.
  • Applied steganographic analysis to extract potential hidden payloads within image files using Steghide and brute-force passphrase attacks.
  • Compiled evidence artifacts and drafted a forensic chain of custody report with mitigation strategies including network segmentation, privilege minimization, and firewall rule enforcement.

Technologies Used

  • FTK Imager
  • Wireshark
  • Autopsy
  • Steghide
  • FileZilla
  • Notepad++
