Hardened Network Services Implementation

Project Overview

  • Deployed and secured DHCP, DNS, and Print Services as part of enterprise infrastructure expansion, emphasizing role hardening and access control.
  • Configured authorized DHCP servers in Active Directory to prevent rogue DHCP attacks and enabled audit logging for scope changes and lease events.
  • Created secure DNS zones with “Secure Only” dynamic updates and DNSSEC validation to mitigate cache poisoning and unauthorized record injection.
  • Implemented RBAC-based printer access controls, hardened Print Spooler service against PrintNightmare exploits, and enforced GPO-based restrictions on remote calls.
  • Used PowerShell, MMC, and WAC for deployment verification and comparative analysis of security posture, auditability, and administrative efficiency.
  • Delivered a hardened, auditable network services stack aligned with Active Directory-integrated security and Microsoft’s Secure Baseline for Windows Server roles.

Environment & Tools

  • Windows Server 2022
  • PowerShell
  • Microsoft Management Console (MMC)
  • Windows Admin Center (WAC)
  • DHCP, DNS, Print Services
  • Active Directory Security Baselines
