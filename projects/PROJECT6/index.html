Active Directory Security Implementation

Project Overview

  • Engineered a secure Active Directory configuration by enforcing strong password, lockout, and account lifecycle policies aligned with NIST and CIS benchmarks.
  • Designed a role-based access control (RBAC) model using AD security groups (Managers, Standard_Users, Contractors) and applied least-privilege principles for all user accounts.
  • Implemented NTFS permission structures with inheritance control, ensuring granular access (Full, Modify, Read) across Confidential, Shared, and Contractors directories.
  • Enabled audit account management and object access logging, capturing event ID 4663 to monitor unauthorized access attempts and policy violations.
  • Hardened AD and file systems using MMC, PowerShell, and Windows Admin Center, integrating auditing and configuration verification into a unified security workflow.
  • Delivered a fully compliant, auditable Active Directory ecosystem with enforced privilege segregation, strong authentication policies, and real-time access visibility.

Environment & Tools

  • Windows Server 2022
  • Active Directory
  • Group Policy (GPO)
  • PowerShell
  • Windows Admin Center (WAC)
  • NTFS Permissions & Audit Logging
